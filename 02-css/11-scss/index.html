Les bases de SASS et SCSS

  • Installer SASS via npm : "npm install -g sass"
  • pour vérifier qu'il est bien installé, lancer "sass" ou "npx sass"
  • Si l'execution des scripts est bloqué, lancer dans powershell en administrateur "set-executionpolicy unrestricted"
  • Les différentes commandes :
    • sass pathToFile.scss pathToFile.css
    • sass --watch pathToFile.scss pathToFile.css
    • sass --watch sassFolder:cssFolder
    • sass --watch sassFolder:cssFolder --style compressed
